Отдельная программа для загрузки прикреплённых файлов из электронной почты на диск.
У нас используется чтобы найти в них счета на оплату Excel.
Есть настройка фильтр по расширениям файлов.


Порядок работы:
1. Изменить файл с настройками settings.txt
2. Запустить DownloadEmailsAttachments.exe 
3. Ждать. Работает бесконечно.
